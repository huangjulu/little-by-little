# 文件目錄

本專案的文件已分類整理，請根據需求查看對應的文件。

## 📁 文件結構

```
docs/
├── README.md                    # 本文件 - 文件索引
├── getting-started/            # 快速開始
│   └── README.md               # 專案快速開始指南
├── architecture/               # 架構設計
│   └── feature-structure.md    # Feature 結構規範
├── api/                        # API 文件
│   └── orders.md               # 訂單 API 文件
├── development/                # 開發指南
│   ├── react-query.md          # React Query 整合說明
│   ├── scripts.md              # NPM Scripts 說明
│   └── performance.md          # 效能優化說明
└── troubleshooting/             # 故障排除
    └── prettier.md             # Prettier 故障排除
```

## 📚 文件分類說明

### 🚀 快速開始 (getting-started/)

**目標讀者：** 新加入的開發者

- **README.md** - 專案快速開始指南
  - 如何安裝依賴
  - 如何啟動開發伺服器
  - 基本專案結構說明

---

### 🏗️ 架構設計 (architecture/)

**目標讀者：** 需要了解專案架構的開發者

- **feature-structure.md** - Feature 結構規範
  - Feature 模組的目錄結構規則
  - 組件層級說明（atoms/molecules/organisms）
  - 單檔命名規則
  - 使用範例

---

### 🔌 API 文件 (api/)

**目標讀者：** 前端開發者、後端開發者、API 使用者

- **orders.md** - 訂單 API 文件
  - API 端點說明
  - 請求/回應格式
  - 錯誤處理
  - 使用範例

---

### 💻 開發指南 (development/)

**目標讀者：** 正在開發功能的開發者

- **react-query.md** - React Query 整合說明

  - 架構概述
  - Query Keys 管理
  - Hooks 使用方式
  - 快取策略

- **scripts.md** - NPM Scripts 說明

  - 開發相關腳本
  - 建置相關腳本
  - 測試相關腳本
  - 格式化相關腳本

- **performance.md** - 效能優化說明
  - 已完成的優化
  - Props 固定參照
  - 最佳實踐

---

### 🔧 故障排除 (troubleshooting/)

**目標讀者：** 遇到問題的開發者

- **prettier.md** - Prettier 故障排除
  - 快速檢查清單
  - 常見問題解決
  - 診斷步驟
